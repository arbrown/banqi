# Pao Rewrite - Planning Documents

This directory contains the planning documents for the rewrite of the Pao game server. These documents are intended to establish a clear set of principles and guidelines for the development of the new application.

6.  [Development Workflow & Conventions](06_DEVELOPMENT_WORKFLOW.md)
7.  [Roadmap & Milestones](07_ROADMAP.md)
