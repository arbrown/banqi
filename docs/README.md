# Ban Qi

`banqi` is a monolithic full-stack application powered by Next.js and Tailwind. It implements a game server for _Ban Qi_.

Ban Qi is a Chinese abstract strategy game that provides a unique blend of chance and strategy. Note that _many_ variations of the rules exist; [Banqi](https://en.wikipedia.org/w/index.php?title=Banqi&oldid=47348070) and [An Introduction to Ban Chi](https://www.thom-boyer.com/ban-chi/rules.html) both describe the specific ruleset recognized by this game server.
